## Inicio Rápido
Con `yarn` instalado.
```bash
yarn install
expo start
```
O usando docker.

- Build de la imagen:
```bash
docker-compose build
```
- Levantar la imagen:
```bash
docker-compose up
```

Luego entrar a http://localhost:19006.
